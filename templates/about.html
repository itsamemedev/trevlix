Ueber TREVLIX

Open Source Algorithmic Trading Intelligence — von der Community, fuer die Community

Mission

TREVLIX wurde entwickelt, um professionelle Krypto-Trading-Technologie fuer jeden zugaenglich zu machen. Waehrend kommerzielle Bots hunderte Euro pro Monat kosten, ist TREVLIX vollstaendig Open Source und kostenlos. Unser Ziel: Die beste selbst-gehostete Trading-KI, die staendig dazulernt.

Mitmachen (Contributing)

TREVLIX lebt von der Community. Jeder Beitrag — ob Code, Dokumentation, Bugfixes oder Feature-Requests — ist willkommen!

So kannst du beitragen:

  1. Forke das Repository auf GitHub
  2. Erstelle einen Feature-Branch: git checkout -b feat/mein-feature
  3. Schreibe Tests fuer deine Aenderungen
  4. Stelle sicher, dass make test und make lint bestehen
  5. Erstelle einen Pull Request mit ausfuehrlicher Beschreibung
# Repository klonen und einrichten
git clone https://github.com/itsamemedev/Trevlix.git
cd Trevlix
make install
make test
