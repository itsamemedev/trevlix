# VERSION

Aktuelle Projektversion: **1.7.1**

Schema: **Semantic Versioning** (`MAJOR.MINOR.PATCH`)

- **Patch** (`x.x.+1`): kleine Fixes, Cleanup, nicht-funktionale Verbesserungen
- **Minor** (`x.+1.0`): neue Module, strukturierende Refactorings, neue Features ohne Breaking Changes
- **Major** (`+1.0.0`): Breaking Changes
