## ūüöÄ Features

- **Multi-Exchange Support** ‚ÄĒ Crypto.com, Binance, Bybit, OKX, KuCoin simultaneously
- **14+ AI Modules** ‚ÄĒ XGBoost, LightGBM, CatBoost, LSTM, Transformer, Random Forest
- **Grid Trading** ‚ÄĒ Automated grid strategies with configurable levels
- **Monte-Carlo Risk Analysis** ‚ÄĒ Portfolio simulations with VaR calculation
- **Telegram Notifications** ‚ÄĒ Real-time alerts for all trades
- **Audit Log** ‚ÄĒ Full action history with timestamps
- **Break-Even Stop-Loss** ‚ÄĒ Automatic SL adjustment after profit
- **Symbol Cooldown** ‚ÄĒ Locks symbols after a loss
- **IP Whitelist** ‚ÄĒ Access control by IP address
- **News Sentiment Filter** ‚ÄĒ Blocks trades on negative news
- **Funding Rate Filter** ‚ÄĒ Avoids expensive short positions
- **Paper Trading** ‚ÄĒ Risk-free testing without real capital
- **Copy Trading** ‚ÄĒ Followers receive all signals in real time
- **2FA** ‚ÄĒ Two-factor authentication (TOTP)
- **Multi-User System** ‚ÄĒ Multiple portfolios on a single instance, each with their own API keys

## ‚ö° Quick Installation

```bash
# One-liner install (Ubuntu/Debian)
curl -O https://raw.githubusercontent.com/itsamemedev/Trevlix/main/install.sh
sudo bash install.sh
```

Or manually:

```bash
git clone https://github.com/itsamemedev/Trevlix.git
cd Trevlix
pip install -r requirements.txt
cp .env.example .env
nano .env          # Enter API keys
python server.py
```

Dashboard: **http://localhost:5000**

## ‚öôÔłŹ Configuration

All settings in `.env`:

```env
# Exchange
EXCHANGE=cryptocom
API_KEY=your_api_key
API_SECRET=your_secret

# Multi-Exchange (optional)
BINANCE_ENABLED=true
BINANCE_API_KEY=...
BINANCE_SECRET=...

# Security
ADMIN_PASSWORD=secure_password
JWT_SECRET=random_string

# Trading
PAPER_TRADING=true   # Always start in paper mode first!
```

Full guide: [INSTALLATION.html](INSTALLATION.html)

## ūüĎ§ Multi-User & API Keys
