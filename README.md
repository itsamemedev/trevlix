## Features

### Trading Engine
- **Multi-Exchange Support** ‚ÄĒ Crypto.com, Binance, Bybit, OKX, KuCoin, Kraken, Huobi, Coinbase
- **9 Voting Strategies** ‚ÄĒ EMA Trend, RSI+Stochastic, MACD, Bollinger, Volume, OBV, ROC, Ichimoku, VWAP
- **Grid Trading** ‚ÄĒ Automated grid strategies with configurable levels
- **Partial Take-Profit** ‚ÄĒ Staged profit taking (25/50/100%)
- **DCA Strategy** ‚ÄĒ Averaging down on falling positions
- **Short-Selling** ‚ÄĒ Bearish trades on futures (Binance/Bybit)
- **Arbitrage Scanner** ‚ÄĒ Cross-exchange price spread detection
- **Paper Trading** ‚ÄĒ Risk-free testing without real capital

### AI & Machine Learning (14+ Models)
- **Random Forest, XGBoost, LightGBM, CatBoost** ‚ÄĒ Ensemble models
- **LSTM Ensemble** ‚ÄĒ Recurrent neural networks for time series
- **Stacking Ensemble** ‚ÄĒ Meta-learner combining all base models
- **Genetic Optimizer** ‚ÄĒ Evolutionary strategy discovery
- **Reinforcement Learning** ‚ÄĒ PPO agent learns from market
- **Anomaly Detection** ‚ÄĒ Isolation Forest stops bot during flash crashes
- **Optuna Hyperparameter Tuning** ‚ÄĒ Bayesian optimization
- **Kelly Criterion** ‚ÄĒ Optimal position sizing based on win probability

### Unique Features
- **Trade DNA Fingerprinting** ‚ÄĒ 7-dimensional fingerprint + pattern mining for each trade
- **Smart Exits** ‚ÄĒ ATR-based volatility-adaptive SL/TP (regime-dependent)
- **Performance Attribution** ‚ÄĒ Hedge-fund-style profit/loss analysis by strategy, regime, session, symbol, sentiment
- **Adaptive Strategy Weights** ‚ÄĒ Self-learning strategy weights with exponential decay and regime-sensitivity
- **KI-Gemeinschaftswissen** ‚ÄĒ Shared knowledge base with optional LLM integration (Ollama, LM Studio)

### Risk Management
- **Circuit Breaker** ‚ÄĒ Automatic trading pause after consecutive losses
- **Drawdown Control** ‚ÄĒ Max portfolio drawdown limit with automatic pause
- **Daily Loss Limit** ‚ÄĒ Max 5% daily loss, resets at midnight
- **Correlation Filter** ‚ÄĒ Blocks correlated positions (>0.75)
- **Liquidity Check** ‚ÄĒ Orderbook spread + minimum volume verification
- **Symbol Cooldown** ‚ÄĒ Locks symbols after a loss
- **Trailing & Break-Even Stop-Loss** ‚ÄĒ Dynamic SL adjustment
- **Monte-Carlo Risk Analysis** ‚ÄĒ Portfolio VaR/CVaR calculation
- **Funding Rate Filter** ‚ÄĒ Avoids expensive short positions
- **Conformal Prediction** ‚ÄĒ Guaranteed prediction intervals

### Market Analysis
- **Fear & Greed Index** ‚ÄĒ Alternative.me sentiment signal
- **Market Regime Detection** ‚ÄĒ Bull/Bear/Range/Crash classification
- **BTC/USDT Dominance Filter** ‚ÄĒ Altcoin trade blocking when BTC dominates
- **News Sentiment** ‚ÄĒ CryptoPanic real-time news as AI signal
- **On-Chain Data** ‚ÄĒ Whale alerts, exchange flows (CryptoQuant)

### Autonomous Agents (NEW v1.5)
- **Auto-Healing Agent** ‚ÄĒ Self-healing system: monitors processes, APIs, DB; auto-recovers on failure with escalation
- **Revenue Tracking Agent** ‚ÄĒ Real PnL tracking after fees, slippage; daily/weekly/monthly summaries; losing strategy detection
- **Multi-Server Control Agent** ‚ÄĒ Cluster management: remote node monitoring, start/stop/deploy, aggregated metrics

### Infrastructure
- **Real-time Dashboard** ‚ÄĒ WebSocket-powered live UI with Socket.IO
- **Full REST API** ‚ÄĒ 85+ JWT-authenticated endpoints
- **Multi-User System** ‚ÄĒ Multiple portfolios, each with own API keys
- **2FA** ‚ÄĒ Two-factor authentication (TOTP)
- **5 Languages** ‚ÄĒ DE, EN, ES, RU, PT
- **Discord & Telegram** ‚ÄĒ Real-time trade notifications
- **GitHub Updater** ‚ÄĒ In-dashboard one-click update & rollback
- **Auto-Backup** ‚ÄĒ Regular data backup with SHA-256 verification
- **Copy Trading** ‚ÄĒ Followers receive all signals in real time
- **Exchange Compatibility** ‚ÄĒ Safe ticker fetching for all exchanges including Crypto.com

## Quick Installation

```bash
# One-liner install (Ubuntu/Debian)
curl -O https://raw.githubusercontent.com/itsamemedev/Trevlix/main/install.sh
sudo bash install.sh
```

Or manually:

```bash
git clone https://github.com/itsamemedev/Trevlix.git
cd Trevlix
pip install -r requirements.txt
cp .env.example .env
nano .env          # Enter API keys
python server.py
```

Dashboard: **http://localhost:5000**

## Configuration

All settings in `.env`:

```env
# Exchange
EXCHANGE=cryptocom
API_KEY=your_api_key
API_SECRET=your_secret

# Security
ADMIN_PASSWORD=secure_password_min_12_chars
JWT_SECRET=random_hex_string_32chars
SECRET_KEY=random_hex_string_32chars
ENCRYPTION_KEY=fernet_key  # python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"

# CORS (optional ‚ÄĒ default allows all origins)
# ALLOWED_ORIGINS=https://yourdomain.com,http://localhost:5000

# Trading
PAPER_TRADING=true   # Always start in paper mode first!

# Unique Features
USE_TRADE_DNA=true           # Trade DNA fingerprinting
USE_SMART_EXITS=true         # Volatility-adaptive SL/TP
USE_ADAPTIVE_WEIGHTS=true    # Self-learning strategy weights

# Notifications (optional)
DISCORD_WEBHOOK=https://discord.com/api/webhooks/...
TELEGRAM_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id
```

## The 9 Trading Strategies

| # | Strategy | Signal Logic |
|---|----------|-------------|
| 1 | **EMA-Trend** | EMA 8/21/50 alignment + price confirmation |
| 2 | **RSI-Stochastic** | Oversold (<35) / Overbought (>65) oscillator |
| 3 | **MACD-Crossover** | Signal line crossover + zero-line filter |
| 4 | **Bollinger Bands** | Mean reversion near band edges + RSI |
| 5 | **Volume Breakout** | High-volume candle in trend direction |
| 6 | **OBV-Trend** | On-Balance-Volume trend confirmation |
| 7 | **ROC-Momentum** | Rate of Change dual-timeframe threshold |
| 8 | **Ichimoku** | Tenkan-Kijun crossover, cloud confirmation |
| 9 | **VWAP** | Volume-Weighted Average Price deviation + RSI |

All strategies vote: **+1** (buy), **-1** (sell), **0** (neutral). Weighted voting determines the final signal.

## Multi-User & API Keys

Each registered user stores their own exchange API keys in the database (Fernet-encrypted). The bot runs trades for each user using their personal credentials ‚ÄĒ no shared keys, full separation. An admin can manage users and global bot configuration via the admin panel.

## Security

- **Fernet Encryption** ‚ÄĒ All API keys encrypted before storage
- **bcrypt Hashing** ‚ÄĒ Secure password storage
- **2FA (TOTP)** ‚ÄĒ Two-factor authentication
- **JWT Authentication** ‚ÄĒ Secure API tokens
- **CSRF Protection** ‚ÄĒ Token validation on forms
- **Rate Limiting** ‚ÄĒ 5 login attempts per IP per 15 min
- **IP Whitelist** ‚ÄĒ Access control by IP
- **Protected Config** ‚ÄĒ Sensitive keys only via `.env` (not API)
- **CORS** ‚ÄĒ Configurable origin restriction
- **Audit Log** ‚ÄĒ Full action history
